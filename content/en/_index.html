{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} The BIOMASS Product Algorithm Laboratory (BioPAL) is an open-source scientific computing project, supporting the development of ESA’s BIOMASS mission algorithms coded in Python. The goal of the BioPAL project is to bridge the gap between advancements in scientific algorithm development and fast integration of these advancements into ESA’s BIOMASS’s ground operations. It is the first time that an official ESA Earth Explorer operational processor is supported by open and collaborative development of processing algorithms within the scope of an open-source software project and community. {{% /blocks/lead %}} {{< blocks/section color="gray-300" >}} {{% blocks/feature icon="fa-lightbulb" title="Tutorials" %}} Go to the tutorials.
